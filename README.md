## 🛑 The Problem
Scattered event data across various sources and the heavy inefficiency of manual tracking. Team members waste valuable time hunting down event details instead of focusing on core tasks.

## 💡 The Solution
An autonomous, end-to-end workflow built with **n8n**. The system regularly scrapes target sources, extracts and seamlessly formats the raw data into **JSON**, and delivers instant, automated notifications directly to the team via WhatsApp.

## 🛠️ Tech Stack
* **Workflow Automation:** n8n
* **Data Extraction:** Web Scraping, REST API
* **Data Processing:** JSON
* **Communication:** Green API (WhatsApp)
